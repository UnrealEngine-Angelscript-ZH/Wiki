



Angelscript中可以通过DebugBreak添加断点

```cpp
DebugBreak();
```

#### 示例：

```cpp
class ADebugExample : AActor
{
    UPROPERTY(DefaultComponent, RootComponent)
    USceneComponent Root;

    UFUNCTION(BlueprintOverride)
    void BeginPlay()
    {
        DebugBreak();
    }
};
```
